#Introduction to Cython

Cython is a C-extension for Python.

Cython tries to combine the ease of Python and speed of C.

It let's you have python syntax with C data types and functions.

The coding examples in this repository are an extension of my blog post: "A Quick Introdcution to Cython"

You can have a look at the blog : <a href='https://wordpress.com/post/83245342/4/'>Quick Introduction to Cython</a>
